Дан список url в json, такого вида

[

{url:"http://google.com", name: "Google"},

{url:"http://yahoo.com", name: "Yahoo"},

{url:"http://bing.com", name: "Bing"},

]

Нужно сделать страницу, которая выводит превью (скриншоты) этих url плиткой. Превью должны генерироваться сами и сразу.

Внизу превью выводить параметр name.

При недоступности url выводить код ошибки на превью.

При клике на превью выводить списком параметры, ассоциированные с этим url (которые хранятся в JSON)
